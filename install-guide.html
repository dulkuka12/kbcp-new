📲 홈 화면에 앱 추가하는 방법

이 앱은 크롬에서 실행한 후 왼 쪽 상단에 있는 ☰ 버튼을 누르면 맨 상단에 홈 화면에 설치 메뉴가 보입니다. 이를 실행하면 자동 설치가 가능합니다.

QR코드로 링크주소를 열었으면 상단 메뉴에서 홈 화면에 설치 메뉴를 실행하면 설치가 됩니다.

삼성 브라우저나 카톡 브라우저로 [기도서] 앱이 열렸으면 홈 화면에 추가 메뉴가 보이지 않습니다. 이때는 링크 주소를 복사하고, 크롬 브라우저을 열어서 주소창에 붙여두기를 해서 기도서를 다시 열면 설치 메뉴가 보입니다.

💡 홈 화면에 추가된 아이콘은 일반 앱처럼 오프 라인에서도 바로 실행할 수 있습니다.
